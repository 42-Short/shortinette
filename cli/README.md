# Shortinette CLI

```console
$ shortinette --help
Easy to use CLI to interact with shortinette

Usage: shortinette [OPTIONS] <COMMAND>

Commands:
  reset-waittime  Sets the waiting time of a student to `0`
  help            Print this message or the help of the given subcommand(s)

Options:
      --base-url <BASE_URL>  Specifies the base URL which is used to send requests to Shortinette [default: http://127.0.0.1:3000]
  -h, --help                 Print help

```
